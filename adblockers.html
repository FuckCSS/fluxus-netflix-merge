Ad Blocker Guide

FluxTV uses third-party embed servers to play content. Many of these servers show ads or pop-ups. An ad blocker makes the experience dramatically better. Use the guide below to find the right one for you.

What device are you on?
Desktop / Laptop
Windows, Mac, Linux — using a browser like Chrome, Firefox, or Edge
Phone / Tablet
iPhone, Android — using Safari, Chrome, or a browser app

Best ad blockers for desktop

uBlock Origin
Recommended
The gold standard. Lightweight, powerful, and fully open source. Works on Chrome, Firefox, and Edge. Blocks ads, pop-ups, and most tracking.

Great
Feature-rich with a clean UI. Available for Chrome, Firefox, Safari, and Edge. Includes extra privacy protections and a stealth mode.

Great

How to install uBlock Origin (Chrome)

1
Go to the Chrome Web Store and search for "uBlock Origin" — or click the link above.
2
Click Add to Chrome, then Add extension in the popup.
3
The uBlock Origin icon will appear in your toolbar. It works automatically — no setup needed.
4
Come back to FluxTV and click Play. If a pop-up still appears, click the uBlock icon and then the power button to toggle it for that page.
Note: Chrome is phasing out Manifest V2 extensions which may affect uBlock Origin. If this happens, switch to Firefox or Brave where uBlock Origin works fully.

Installed FluxTV as a web app?

If you added FluxTV to your home screen, it opens in a standalone browser window. Ad blocking depends on which browser handled the install.

1
iPhone: Install AdGuard from the App Store and enable it under Settings > Safari > Extensions. Then add FluxTV to your home screen from Safari.
2
Android: Install Brave, open FluxTV in Brave, then tap the menu and select Add to Home Screen. Brave's shields will stay active.
Note: On iPhone, ad blocking in installed web apps is limited. For the best experience, open FluxTV directly in Brave for iOS rather than using the home screen shortcut.
